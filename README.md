# Анализ конверсии онлайн-записей в сети медицинских клиник

## 📌 Описание проекта
В сети медицинских клиник наблюдалась проблема: пользователи значительно чаще записывались на приём по телефону, чем через онлайн-форму на сайте.

Это создавало дополнительную нагрузку на колл-центр и снижало эффективность цифровых каналов.

## 🎯 Цель проекта
Увеличить долю онлайн-записей за счёт анализа пользовательского поведения и улучшения пользовательского интерфейса.

## 🔍 Задачи
- Проанализировать пользовательский путь на сайте
- Построить воронку записи на приём
- Выявить узкие места (bottlenecks) в конверсии
- Исследовать поведенческие метрики страницы услуги
- Сформулировать UX-гипотезу
- Оценить эффект от изменений

## 🛠 Используемые инструменты
- SQL (анализ данных, построение воронки)
- Python (pandas, matplotlib)
- Jupyter Notebook

## 📊 Ожидаемый результат
- Выявлены причины низкой доли онлайн-записей
- Предложены изменения в интерфейсе
- Зафиксирован рост конверсии в онлайн-запись (~130%)

## 📁 Структура проекта
- `data/` — исходные данные
- `sql/` — SQL-запросы
- `notebooks/` — анализ и визуализации
